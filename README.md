# Project built with Javascript

![screenshot](https://github.com/binhtranhuu/project-javascript-sun/blob/master/frontend/images/demo-javascript.png)

## Features

- User Json Server fake API
- Use ajax load product in frontend
- Full featured shopping cart
- Product pagination
- Product filter by category

## Usage

### Install Dependencies (frontend & json server)

```
$ npm install
$ cd frontend
$ npm install
```

### Run Json Server

```
$ npm start
```

### Run Frontend

```
# open new terminal
$ cd frontend
$ npm start
```
