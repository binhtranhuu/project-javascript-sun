## Features

- User Json Server fake API
- Use ajax load product in frontend
- Full featured shopping cart
- Product pagination
- Product filter by category

## Usage

### Install Dependencies (frontend & backend)

```
$ npm install
$ cd frontend
$ npm install
```

### Run Json Server

```
$ npm start
```

### Run Frontend

```
# open new terminal
$ cd frontend
$ npm start
```
